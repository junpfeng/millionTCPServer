# millionTCPServer

## 网络数据包

### 基本包格式

报文有两部分：`包头`和`包体`

* 包头：描述本次数据包的大小等信息
* 包体：具体的数据

### 常见问题

定长数据/边长数据、粘包/分包

* **粘包**：多个数据包由于各种原因被接收端一起接受成了一个大包，就像多个数据包粘在一起。此时，就需要根据包头定义的数据长度进行拆包。
* **分包**：当某个数据过长，被分为多个包发送，接收端要接受完所有分包的数据。
